# 요약: 01-MCP-Tools.ipynb

## 1. MCP(Model Context Protocol) 개요
- MCP는 AI 애플리케이션과 외부 도구(API, DB, 파일 등) 간 상호작용을 표준화한 오픈 프로토콜입니다.
- LLM이 다양한 외부 시스템과 일관된 방식으로 통신할 수 있도록 표준 인터페이스 계층을 제공합니다.

## 2. MCP의 핵심 특징
- 표준화된 도구 인터페이스: 모든 외부 도구를 동일한 프로토콜로 연결.
- 다양한 전송 메커니즘: stdio(로컬), Streamable HTTP(원격) 지원. HTTP+SSE는 2025년 3월 이후 폐기 예정.
- 동적 도구 검색: 런타임 시 사용 가능한 도구 자동 탐색.
- 확장 가능한 아키텍처: 모듈형 구조, 멀티 서버 지원.

## 3. MCP 서버 유형 및 실습
- 로컬 서버(stdio): 빠르고 안정적, 파일 관리 등 로컬 작업에 적합.
- 원격 서버(Streamable HTTP): 네트워크 기반, 외부 API 연동 및 실시간 스트리밍에 적합.
- 검색 서버(stdio+RAG): 대용량 문서 의미 기반 검색, 벡터 유사도 검색 지원.

## 4. MultiServerMCPClient
- 여러 MCP 서버를 하나의 통합 인터페이스로 관리하는 컴포넌트.
- 서버별 도구를 자동 탐색·통합, 장애 복구 및 에러 처리 지원.

## 5. 실습 코드 예시
- 서버별 Python 코드 예시와 MCP 클라이언트 설정법, LLM과 MCP 도구를 통합한 React Agent 생성법 등 실습 중심으로 구성.

## 6. React Agent와 MCP 통합
- React Agent는 추론(Reason)과 행동(Act)을 반복하는 패턴을 구현.
- MCP 도구와 함께 사용하면 강력한 에이전트를 만들 수 있음.
